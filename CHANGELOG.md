# 更新日志

所有版本更新记录。

---

## v1.1.0 (2026-07-22)

### 新增
- 新增 `max_free_pack_count` 配置项，多图低于此数量时逐张发送，高于则自动打包
- logger 控制台输出下载进度（"正在下载第x张图片，共x张"）

### 优化
- 遵循 AstrBot 插件市场规范，统一 desc/version 格式
- 所有配置项添加 hint 说明字段
- 请求头精简，去掉多余的 Referer/Accept

### 修复
- 修复多图探测逻辑，改用图床 `-999.jpg` 响应判断页数，无需 Ajax API
- 清理多余辅助方法，精简代码

---

## v1.0.0 (2026-07-15)

### 新增
- 首次发布
- 从 pixiv.re 等第三方图床下载 Pixiv 作品
- 单图直发 JPG，多图打包 PDF/ZIP 发送
- 支持三个图床源切换：pixiv.re / pixiv.cat / i.pixiv.re
- 可选本地保存图片/打包文件
- 简洁输出，不展示作品信息，直接发图
